## Run Locally

Clone the project

```bash
  git clone https://github.com/govardhanob/online-learningwebsite.git
```

Go to the project directory

```bash
  cd Classroomproject
```

Install dependencies

```bash
  pip install -r requirements.txt

```

Start the server

```bash
  python manage.py runserver

```
